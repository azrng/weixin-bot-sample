# 本次目标

完成微信官方 iLink Bot API 演示 Demo 一期实现，支持微信绑定、主动推送消息和用户发消息后的固定祝福语回复。

# 核心改动

- 新建 `.NET 10 + Blazor Server` Demo 项目与解决方案。
- 接入 `BootstrapBlazor` 与 `QRCoder`，完成页面主题、组件根节点和应用图标。
- 实现 `WeixinBotDemoService`，复用 `OpenCowork` 中官方微信渠道协议思路，支持：
  - 获取二维码 `get_bot_qrcode`
  - 轮询绑定状态
  - 长轮询接收消息 `getupdates`
  - 主动推送与自动回复 `sendmessage`
- 实现本地 JSON 状态持久化，保存配置、绑定状态、最近消息、回复记录和日志。
- 实现单页演示页面，覆盖基础配置、二维码绑定、监听控制、主动推送、最近消息和操作日志。
- 新增单元测试，验证固定祝福语服务行为。
- 新增仓库级 `NuGet.config`，固定使用 `nuget.org`，避免本机无权限私有源影响还原。
