# 本次目标

- 修复 Demo 工作台页面高频轮询、状态高频写盘、媒体处理内存占用和后台任务生命周期不完整的问题。

# 核心改动

- 将页面层从固定 2 秒轮询改为订阅 `WeixinBotDemoService` 的状态变更事件，避免多标签页持续后台刷新。
- 重写 `JsonStateStore`，改为合并写入、延迟落盘和原子替换文件，减少 JSON 全量重写频率。
- 将媒体发送从内存字节数组改为临时文件 + 流式加密上传；将媒体下载改为流式解密写入缓存文件。
- 为 `WeixinBotDemoService` 增加 `IAsyncDisposable`，补齐绑定/监听后台任务的取消、等待和释放，并收紧启动竞态窗口。
- 新增 `JsonStateStore` 测试，并把本地验证目录 `.verify/` 加入忽略规则。
